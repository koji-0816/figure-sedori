# フィギュアせどり判定プロトタイプ ご案内

## URL
https://figure-sedori.onrender.com

## できること
- ヤフオクの検索結果を自動取得（商品名・画像・現在価格・送料・残り時間）
- 買取価格を入力すると、見込利益を自動計算して判定表示

## 使い方
1. キーワード欄に商品名を入力（例: 一番くじ ワンピース）
2. 「検索」ボタンを押す
3. 一覧の各商品の「買取価格（手入力）」欄に、確認した買取相場を入力
4. 自動で「見込価格」と「判定」が表示される（緑色の行は利益見込みあり）

見込価格の計算式：
```
見込価格 = 買取価格 ÷ 1.2 − 送料
利益 = 見込価格 − 現在価格
```

## 現時点の制約（プロトタイプ版）
- **買取価格は手入力**です。自動取得はしていません。買取価格検索サイト側の利用規約で、営利目的でのデータ自動抽出が禁止されているためです。
- 送料が「要確認」と表示される商品があります（ヤフオク側の表示形式のばらつきによるもの。0円として計算しているため、実際の送料を確認してください）
- 検索結果は1ページ目まで（ページ送り未対応）
- 無料サーバーのため、しばらくアクセスがないとスリープします。その場合、初回アクセス時に表示まで数十秒かかることがあります。デモ前に一度アクセスして起こしておくことを推奨します。
- LINEなどへの自動通知は未実装
